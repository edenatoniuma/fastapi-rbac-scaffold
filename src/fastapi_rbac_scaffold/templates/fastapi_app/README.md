# {{ project_name }}

FastAPI service generated by `fastapi-rbac-scaffold`.

## Quick Start

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -e ".[dev]"
copy .env.example .env
docker compose up -d postgres redis
alembic upgrade head
app-seed-iam
app-bootstrap --username admin --password change-me --email admin@example.com
uvicorn app.main:app --reload --port 8000
```

`app-bootstrap` creates or updates the first platform admin. Platform admin APIs are protected by `require_platform_admin`; tenant APIs are protected by `require_permission("<module>:<resource>:<action>")`.

## Included

- FastAPI app factory
- async SQLAlchemy
- Redis client lifecycle
- Alembic migrations
- JWT authentication
- multi-tenant RBAC model
- permission dependency helpers
- unified response models

## Auth Boundary

- Middleware only checks that protected routes have a bearer token.
- `get_current_principal` validates JWT, Redis blacklist, user status, tenant membership, and permission cache.
- `require_permission("iam:user:list")` protects route-level access.
- Service helpers should enforce object-level tenant access for business resources.
